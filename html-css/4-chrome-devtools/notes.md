# Chrome Web DevTools

- It is used to inspect the `HTML,CSS and JS`  in a website.
- The `Inspect Element` tool can be clicked and click the element you want to inspect to see its html and styles.

- The `Styles` section shows all the *styles applied* to the element.

- The `Computed` section shows the final `Box model` of the element.

## Css Box Model

- Margin : space outside the element
- Padding: space inside the element

If we set height and width explicitely, 
When the content gets bigger, it overflows.

Therefore padding and width might be a good choice.
